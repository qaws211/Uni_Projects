# Progetto: Analisi delle Serie Storiche con SVM 

## Descrizione del Progetto
Questo progetto si propone di utilizzare la tecnica del Support Vector Machine (in particolare la Support Vector Regression) per prevedere i ricavi di Apple Inc. al primo trimestre 2022.
Il dataset contiene i seguenti dati trimestrali dal 31/12/1990 al 31/12/2021:

-Ricavi
-Fiducia_Consumatori_Usa
-Consumer Loans
-IPI Semiconductor
-Copper
-Delinquency Rate
