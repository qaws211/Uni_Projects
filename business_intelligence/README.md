## UEFA Champions League Analysis  
**Tool:** Microsoft Power BI  

### Description  
This project analyzes the history of the **UEFA Champions League** from 1955 to 2023 and provides an in-depth analysis of the 2021/2022 season.  
The goal is to offer an interactive and comprehensive overview of the competition, highlighting the performance of teams, players, and coaches.

### Main datasets  
- UEFA Champions League winners list (1955‚Äì2023)  
- Historical statistics for teams, countries, players, and coaches  
- Detailed data for the 2021/2022 season  

### Output  
- Champions League historical overview dashboards  
- Historical analysis (1955‚Äì2023)  
- Detailed analysis of the 2021/2022 season  
- Team classification using **K-Means clustering**  
- Mobile-optimized dashboards

## Billionaires & Wealth Analysis  
**Tool:** Qlik  

### Description

### Main datasets

- Relationship between the number of billionaires and economic indicators

## Bank Marketing Campaign Analysis  
**Tool:** Tableau  

### Description  
This project analyzes a **direct marketing campaign** carried out by a Portuguese bank, aimed at promoting the subscription of term deposits via phone calls.  
The objective is to identify customer profiles that are more likely to accept the offer and to evaluate campaign effectiveness.

### Dataset  
- Bank Marketing Dataset (UCI Machine Learning Repository)  

### Objectives  
- Identify the most promising customer segments  
- Evaluate the effectiveness of the marketing campaign  
- Support strategic decisions for future campaigns  

### Output  
- Customer Overview  
- Campaign Performance Analysis  
- Customer behavior analysis and **Buyer Persona** definition
